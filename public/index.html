Restaurant Management System API

Welcome to the Restaurant Management System API. This API provides endpoints for managing restaurants, food items, categories, and user authentication.

Get started by exploring the API endpoints and data models.

API Documentation

API Endpoints

Below are the main endpoints provided by the API:

  • Authentication
    • POST /register
    • POST /login
  • Restaurants
    • POST /restaurants
    • GET /restaurants
    • GET /restaurants/{id}
    • PUT /restaurants/{id}
    • DELETE /restaurants/{id}
  • Foods
    • POST /foods
    • GET /foods
    • GET /foods/{id}
    • PUT /foods/{id}
    • DELETE /foods/{id}
  • Categories
    • POST /categories
    • GET /categories
    • GET /categories/{id}
    • PUT /categories/{id}
    • DELETE /categories/{id}
  • Users
    • GET /users/{id}
    • PUT /users/{id}
    • PUT /users/update-password/{id}
    • PUT /users/reset-password/{id}
    • DELETE /users/{id}
